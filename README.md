# Котокафе

## Правила игры (кото-кафе)
```
1) Каждому игроку выдается лист с башнями.
2) Берем столько кубиков какое количество игроков (от 2 до 4) + 1
3) Первый игрок кидает кубики, после он берет один из кубиков себе. Далее кубик выбирает следующий по счету игрок.
   (В следующий ход, первым будет выбирать кубик следующий по счету игрок и так по кругу)
4) На столе останется один кубик который будет 'общим'.
5) В итоге у каждого игрока есть два кубика. Один из них должен указывать предмет, который игрок будет распологать на
   башне, другой место на этой башне.
6) Игра продолжается до тех пор пока один из игроков обустроит три башни, после чего подсчитывают баллы и у кого их 
   больше побеждает.
```
## Текстовый интерфейс

предметы:1-домик, 2-клубок, 3-бабочка, 4-миска, 5-подушка, 6-мышка

Два игрока: George, Dmitry
```
cubes: 1 5 3
George: выбирает 5
Dmitry: выбирает 3
central cube: 1
George: 5-место, 1-предмет(можно выводить таблицу с данными о заполнении башен)
Dmitry: 1-место, 3-предмет
-----
cubes: 6 3 2
Dmitry: выбирает 6
George: выбирает 2
central cube: 3
Dmitry: 3-место, 6-предмет
George: 3-место, 2-предмет
-----
cubes: 5 6 5
George: выбирает 6
Dmitry: выбирает 5
central cube: 5
George: 5-место, 6-предмет
Dmitry: 5-место, 5-предмет
-----
cubes: 5 4 3
Dmitry: выбирает 5
George: выбирает 3
central cube: 4
Dmitry: 4-место, 5-предмет
George: 4-место, 3-предмет
-----
cubes: 2 4 6
George: выбирает 6
Dmitry: выбирает 4
central cube: 2
George: 2-место, 6-предмет
Dmitry: 2-место, 4-предмет
-----
.....
-----
Dmitry WIN!!!
```
